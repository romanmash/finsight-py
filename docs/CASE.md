# CASE

FinSight delivers operator decision support through coordinated Python agents.

## User Surfaces

- Telegram for user interaction
- Dash console for operator monitoring and control

## Core Flow

1. Request enters API
2. Manager routes mission through LangGraph
3. Specialist agents execute bounded responsibilities
4. Data/tools are accessed through MCP servers
5. Result is persisted and returned to user channel

## Operational Objectives

- Deterministic, auditable agent execution
- Config-driven runtime behavior
- Complete cost and trace visibility
- Reliable offline testability
